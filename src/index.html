Press to start drawing, experiment by Jorge Toloza x Codrops

Explore the stop-motion crayon effect featured in a Codrops article by Jorge Toloza. Created using
p5 brush, a versatile library in the p5.js ecosystem powered by #WebGL, this effect transforms mouse movements into dynamic, hand-drawn animations. Start drawing by simply pressing and holding the mouse button while
moving across the canvas, producing organic and playful visuals.
